# tt-yaml-upgrade-tool

Tiny Tapeout Project Yaml V4 to V6 upgrade tool

## Development

You need to have Node.js 18 or newer installed on your machine.

1. Clone this repo
2. Run `npm install`
3. Run `npm start`

Enjoy!
